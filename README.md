# econ1626-labour-policy-proposal

This repository contains my Assessment 2 policy proposal for ECON1626 Economics of Artificial Intelligence.

## Files

- `proposal.md` contains the 1,200-word policy proposal.
- `README.md` explains the repository structure and workflow.

## Topic

The proposal examines how Australia can move from AI experimentation to shared productivity through a workplace AI adoption plan. It focuses on firm-led adoption, role-specific worker training, task redesign and worker safeguards.

## Workflow

This project is written in Markdown and managed through GitHub version control. Commits are used to record the development of the proposal from outline to final submission.

## Use of AI tools

AI tools were used to support brainstorming, structure planning, evidence mapping, Markdown formatting and editing checks. The final argument, source selection and policy judgement were reviewed and controlled by the author.
